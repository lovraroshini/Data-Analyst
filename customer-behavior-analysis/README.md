# Customer Behavior Analysis
### RISE Internship Project | Data Science & Analytics

---

## About the Project
This project analyzes 6000 real e-commerce customer records to understand
customer behavior, identify spending patterns, segment customers into groups,
and flag at-risk customers who might stop buying.

---

## Tools Used
- Python - Programming language
- Pandas - Data loading and cleaning
- NumPy - Mathematical calculations
- Matplotlib - Data visualization
- Seaborn - Data visualization
- Scikit-learn - Customer segmentation
- Jupyter Notebook - Development environment

---

## Files
- customer_behavior_analysis.ipynb - Main project notebook
- ecommerce_customer_features.csv - Dataset

---

## What I Did
- Loaded and cleaned 6000 customer records
- Analyzed satisfaction scores and order values
- Segmented customers into 3 groups using KMeans
- Identified high and low spenders
- Flagged at-risk customers

---

## Key Findings
- Average satisfaction score: 8.1 / 10
- Average order value: 80.49
- Average total orders: 8.6
- Loyalty members: 18.1%
- Highest order value: 1006.53
- Lowest order value: 10.00

---

## Recommendations
1. Send special offers to at-risk customers
2. Reward high spenders with loyalty perks
3. Investigate low satisfaction scores
4. Convert low spenders into frequent buyers
